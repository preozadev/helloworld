# ASCII Boat

โปรแกรม Python สำหรับแสดงภาพเรือด้วย ASCII Art บน Terminal

## ตัวอย่างผลลัพธ์

```text
                   |\
                  /| \
                 /_|__\
               __|____|__
         _____/__________\_____
         \                    /
          \__________________/
           \                /
            \______________/
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
```

## วิธีใช้งาน

1. ติดตั้ง Python 3
2. บันทึกโค้ดเป็นไฟล์ `boat.py`
3. รันโปรแกรม

```bash
python boat.py
```

## โค้ดตัวอย่าง

```python
print('hello')

text = r'''

                   |\
                  /| \
                 /_|__\
               __|____|__
         _____/__________\_____
         \                    /
          \__________________/
           \                /
            \______________/
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

'''

print(text)
```
